# Password Manager

A simple Python desktop application that helps users generate, save, and retrieve passwords for different websites.

The project uses Tkinter for the graphical user interface and JSON for persistent data storage.

---

## Features

* Generate random secure passwords
* Save website, email, and password information
* Store account data in a JSON file
* Search for saved accounts by website name
* Handle missing files using exception handling
* Display stored account details through message boxes
